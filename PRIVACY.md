## Privacy Policy for VoxTab Premium TTS
VoxTab is built with a strict local-first architecture.

Data Collection: VoxTab does not collect, store, or transmit any personally identifiable information, web browsing history, or user data.

Website Content: Text highlighted by the user is processed strictly locally on the user's device using the browser's native Web Speech API. No text or data is ever sent to external servers.

Permissions: All permissions (storage, activeTab, contextMenus, and host permissions) are used exclusively to provide the core text-to-speech functionality locally on your machine.
